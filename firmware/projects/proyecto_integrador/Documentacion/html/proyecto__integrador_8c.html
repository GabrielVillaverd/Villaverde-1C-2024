◆ AnalogicoAdigital()

void AnalogicoAdigital ( void * pvParameter)

Tarea encargada de convertir los datos analogicos (de luminocidad) a valores degitales a finde poder trabajar con estos datos.

Los valores analogicos que ingresan a CH3 pasan a digitales y se guaran en ValorAnalogico

Enviamos valores medidos de luminocidad por bluettoth

En caso de que el parametro de luz sea mayor o igual al valor seteado se apagan las luces y se abren las cortinas

En caso de que el parametro de luz sea menor al valor seteado se prenden las luces y se cierran las cortinas

◆ Calefaccion()

void Calefaccion ( void * pvParameter)

Tarea encargada de medir temperatura mediante sensor Si7007 y enviarlos por bluetooth.

Promedio de 3 valores de temperatura medidos

Se envian por bluetooth los valores de humedad medidos

Led que nos avisa que la temperatura esta dentro del rango aceptado

Si la temperatura es menor al umbral de temperatura tolerado se prende el led 3 simulando aire en calor y se apaga el led que simula temperatura en el rango aceptado

Si la temperatura es menor al umbral de temperatura tolerado se prende el led 3 simulando aire en calor y se apaga el led que simula temperatura en el rango aceptado
